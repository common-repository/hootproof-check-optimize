== Description ==

***Für eine deutsche Beschreibung bitte die [Plugin-Website](https://hootproof.de/hootproof-plugin) besuchen oder auf den Tab "Deutsch" klicken.***

The HootProof Check & Optimize plugin checks your WordPress installation and suggests improvements and optimizations. It aims at supporting you to make your website 
* faster
* more secure
* search-engine optimized
* compliant with EU and German privacy laws


The plugin checks more than 50 potential issues on your site from the areas of security, website performance, SEO, user privacy, and legal requirements. On the plugin dashboard you will find all the optimization suggestions that are applicable to your site and a link to a detailed description on how to implement them. 

= GTmetrix Integration =

The plugin also provides full GTmetrix integration, so you can test your site performance from our WordPress admin area and evaluate the results. HootProof is not affiliated with GTmetrix in any way.

= Comment IP removal =

Also, the plugin automatically removes IP addresses from new user comments and provides the option to delete all the IP addresses of existing comments. This is to comply with privacy laws that forbid tracking personal data without the user’s awareness.

= Show your users that you care =

If you want to, you can also show a small HootProof logo in your footer, demonstrating to your users that your site respects privacy laws and strives for optimization.

= It doesn’t affect your site performance =

Apart from IP comment removal, the plugin only works in the admin area so it does not affect the performance of your site.


== Installation ==

Provide installation instructions for the plugin.

After installation and activation, the basic plugin configurations are set by default. In order to user the performance tests, you need a free Gtmetrix account. Go to [GTmetrix.com](http://gtmetrix.com) to register and retrieve your API key.

== Deutsch ==

[Zur ausführlichen Dokumentation zur Installation und Einrichtung des Plugins](https://hootproof.de/plugin-dokumentation).

Das HootProof Plugin hilft dir, deine WordPress-Installation gründlich zu prüfen und zu optimieren. Es überprüft die Einstellungen, Plugins und den Code deiner WordPress-Seite und schlägt dir Optimierungen vor. Teste die Performance deiner Seite, verbessere den Datenschutz für deine Besucher und erfahre, wie du deine Seite noch sicherer und suchmaschinenoptimiert machen kannst.

Dazu gehören über 50 Checks und Vorschläge aus den Bereichen
*Performance & Website-Geschwindigkeit
*Suchmaschinenoptimierung (SEO)
*Sicherheit
*Datenschutz und Rechtliches (mit Fokus auf den deutschen Datenschutz und das deutsche Online-Recht)

Die Checks werden nur im Admin-Bereich ausgeführt, d.h. das Plugin beeinträchtigt selbst nicht die Geschwindigkeit deiner Website.

= Entfernen von IP-Adressen bei Kommentaren =

Das Plugin entfernt bei neuen Kommentaren die IP-Adressen des Kommentierenden. Das Speichern der vollen IP-Adressen ohne Einverständnis ist datenschutzrechtlich bedenklich. Dieses Feature ist deaktivierbar.
Die IP-Adressen bestehender Kommentare können mit einem Klick entfernt werden.

= GTmetrix Integration =

Das Plugin integriert GTmetrix, um die Performance und Geschwindigkeit der Seite zu prüfen und detaillierte Vorschläge zum Beschleunigen und Optimieren zu machen. Starte Performance-Tests direkt aus deinem WordPress-Adminbereich und analysiere die Ergebnisse.

Um die Performance-Tests nutzen zu können, benötigst du ein kostenloses Konto bei [GTmetrix](http://gtmetrix.com)

= Empfohlene Plugins=
Das HootProof-Plugin schlägt dir anhand deiner aktiven Plugins weitere kostenlose Plugins vor, mit denen du deine Seite noch besser machen kannst.

= Zeig deinen Besuchern, dass deine Seite “hootproof” ist =

Wenn du möchtest, kannst du im Footer deiner Seite einen kleinen HootProof-Icon integrieren, der deinen Besuchern zeigt, dass du Datenschutz, Sicherheit und Performance ernst nimmst und sie bei dir gut aufgehoben sind. Zur Aktivierung des HootProof-Icons gibt es eine Checkbox in den Plugin-Einstellungen.
